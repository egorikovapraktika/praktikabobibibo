ПОЯСНИТЕЛЬНАЯ ЗАПИСКА
======================

1. НАЗНАЧЕНИЕ СИСТЕМЫ
---------------------
Система предназначена для автоматизации производственных процессов на предприятии 
по производству средств защиты растений. Система включает три модуля:

- Модуль технолога - для разработки рецептур, техкарт и производственных заданий
- Модуль лаборатории - для контроля качества сырья и готовой продукции
- Модуль аппаратчика - для исполнения технологических операций

2. АРХИТЕКТУРА СИСТЕМЫ
-----------------------
- Клиент-серверная архитектура
- База данных: Microsoft SQL Server
- Клиенты: WPF приложения (.NET 5.0)
- API: REST API (ASP.NET Core 5.0)

3. ОСНОВНЫЕ ФУНКЦИИ
-------------------
3.1 Модуль технолога:
  - Управление продукцией
  - Создание и редактирование рецептур
  - Разработка технологических карт
  - Формирование производственных заказов
  - Запуск и мониторинг партий
  - Фиксация отклонений

3.2 Модуль лаборатории:
  - Контроль качества партий
  - Проведение испытаний
  - Автоматическая проверка нормативов
  - Принятие решений (одобрение/браковка)
  - Формирование протоколов

3.3 Модуль аппаратчика:
  - Просмотр активных партий
  - Пошаговое выполнение техпроцесса
  - Ввод фактических параметров
  - Фиксация отклонений

4. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
-------------------------
- ОС: Windows 10/11
- .NET 5.0 Runtime
- SQL Server 2016+
- ОЗУ: не менее 4 ГБ
- Диск: не менее 500 МБ

5. ИНСТРУКЦИЯ ПО УСТАНОВКЕ
--------------------------
1. Установить .NET 5.0 SDK
2. Развернуть базу данных из скрипта
3. Запустить API: dotnet run
4. Запустить приложение: dotnet run
